# SyncTalk

A modern real-time chat application built using React, Node.js, Express and Socket.IO.

## Features
- Real-time messaging
- Instant updates using WebSockets
- Responsive UI
- Multi-user support

## Tech Stack
- React
- Vite
- Tailwind CSS
- Node.js
- Express.js
- Socket.IO

## Installation

### Client
cd client
npm install
npm run dev

### Server
cd server
npm install
npm run dev
